# 木構造による文法構造の表現

C ではほとんどの演算子が左結合
代入演算子は右結合

四則演算

```
expr = mul ("+" mul | "-" mul)*
mul = primary ("*" primary | "/" primary)*
primary = num | "(" expr ")"
```

> 再帰下降構文解析法でパーサを書く時の基本的な戦略は、これらの非終端記号一つ一つをそのまま関数一つ一つにマップするというものです。
